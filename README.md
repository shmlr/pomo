# PomoDag — Workflow & Prayer Control ⏱️🕋

**https://shmlr.github.io/pomo/** — это кастомный инструмент для высокопродуктивной работы, созданный для синхронизации бизнес-задач с временем намаза. В телефоне пользуйтесь планировщиком как приложением, нажмите в браузере "Поделиться" -> "На экран Домой".

Обновленная инструкция PomoDag v3.9
1. Как включить уведомления (Важно!)
При каждом открытии страницы нажимай красную кнопку «🔔 ВКЛЮЧИТЬ УВЕДОМЛЕНИЯ».

Это активирует звук и дает браузеру «зеленый свет» на отправку системных сообщений.

Если появится запрос на разрешение уведомлений — жми «Разрешить».

2. Настройка повторов (Твоя главная фишка)
Теперь при добавлении задачи ты можешь выбрать один из трех режимов:

📌 Один раз: Задача исчезнет из списка сразу после того, как таймер дойдет до нуля. Идеально для разовых созвонов или быстрых дел.

🔄 Каждый день: Задача останется в списке навсегда. Каждое утро она будет «обнуляться» и снова ждать своего времени. Подходит для ежедневных проверок склада или ответов в Direct.

📅 Будни (Пн-Пт): Умный режим. Уведомление сработает только в рабочие дни. В субботу и воскресенье планировщик тебя не потревожит, давая время на отдых с семьей.

3. Настройка времени намаза (⚙️)
Если время в твоей мечети отличается от стандартного для Махачкалы:

Нажми на шестеренку ⚙️.

Введи разницу в минутах (например, 2 или -1) для нужной молитвы.

Нажми на шестеренку еще раз, чтобы скрыть панель. Настройки сохранятся в памяти браузера.

4. Работа «Поверх всех окон»
Тебе не нужно постоянно смотреть на вкладку. Сверни браузер и занимайся контентом или аналитикой.

В нужное время в углу экрана (Windows/macOS/Android) появится уведомление.

Клик по уведомлению сразу перенесет тебя к таймеру задачи.

5. Конфиденциальность и доступ
Все данные локальны: Твой список задач не видит никто, кроме тебя. Он хранится в памяти твоего текущего устройства (телефона или ноутбука). Если ты откроешь ссылку с другого устройства, список будет пуст — это гарантирует, что твои бизнес-планы видишь только ты.
