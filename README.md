# PomoDag — Workflow & Prayer Control ⏱️🕋

**PomoDag** — это кастомный инструмент для высокопродуктивной работы, созданный для синхронизации бизнес-задач с временем намаза. 

Идеально подходит для управления контент-планами **Berbena**, маркетинговыми стратегиями **Palleta** и разработкой цифровых продуктов (например, **Finance PDF Kit 2026**).

## 🚀 Основные возможности
* **Precision Alerts:** Мгновенные уведомления ровно в минуту начала намаза или рабочей сессии.
* **Business Focus:** Настройка задач под разные циклы (разовые задачи, ежедневные рутины или рабочие будни Пн-Пт).
* **Makhachkala Sync:** Автоматический расчет времени молитв (MWL Method) с возможностью ручной корректировки через ⚙️.
* **Full Privacy:** Данные о ваших задачах и финансах не уходят на серверы, а хранятся локально в вашем браузере.

## 🌐 Быстрый старт
1. Перейдите по ссылке: `https://shmlr.github.io/pomo/`
2. **Активация звука:** Обязательно нажмите кнопку **"🔔 Нажми здесь для звука"** при первом открытии, чтобы браузер разрешил аудио-уведомления.
3. Добавьте страницу на главный экран телефона как веб-приложение.

## 🛠 Стек технологий
* **Frontend:** Чистый HTML5, CSS3 (Light Mode), Vanilla JavaScript.
* **API:** Aladhan (Timings by City).
* **Storage:** LocalStorage API.
